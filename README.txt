For Single Linked List implementation in CSharp go to CSharp folder.
Node.cs -> Single linked list node is created here.
SingleLinkedList.cs -> Includes the logics for all the implementations.
Demo.cs -> Includes the method calling of the logics present in the above file.

For Double Linked List implementation in CSharp go to DoubleLinkedList folder.
Node.cs -> Double linked list node is created here.
DoubleLinkedList.cs -> Includes the logics for all the implementations.
Demo.cs -> Includes the method calling of the logics present in the above file.

You can ignore the rest of the files.
